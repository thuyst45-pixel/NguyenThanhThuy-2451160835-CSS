Kết quả mong đợi:

Element font-size color Giải thích color
"Sản phẩm A" (h2.title.highlight trong #featured) 20px green .highlight { color: green !important } thắng mọi rule khác
"Mô tả sản phẩm" (p trong #featured .card) inherited (14px từ .container) blue .card p { color: inherit } → kế thừa từ .card → blue
"Sản phẩm B" (h2.title) 20px blue Kế thừa color từ .card (blue), không có rule trực tiếp set color cho .title
"Mô tả sản phẩm B" (p.highlight) inherited (14px từ .container) green .highlight { color: green !important } thắng .card p { color: inherit }

Mở DevTools (F12) → Inspect từng element để xác minh.
